# Ejecución tarea #2
## Ejecución del código
Ejecute el código de la carpeta **Solucion** mediante el siguiente comando en la terminal en caso de trabajar en Linux:
> make

Es importante tener instalada la herramienta de sintesis de Yosys antes de correr el programa.
Finalmente, el ejemplo de la solución se encuentra en el documento **Enunciado_tarea_#2.pdf**.
